# Tradebot — App & Strategy Review

**Date:** 2026-07-04 · **Status:** all paper/simulated — no order-placement code exists anywhere in the repo.

---

## 1. The app

A layered research + paper-trading system for NSE (NIFTY 50 universe, 48 stocks + index benchmark, 1,240 trading days 2021-06-23 → 2026-06-25, daily OHLCV via yfinance, price/volume only).

```
 L0  Shared base       config.py · data_io.py
 L1  Signals           lowvol · momentum · strategy(SMA)
 L2  Backtest engine   backtest · backtest_lowvol/momentum · strategy_base
 L3  Paper simulators  paper_trader · options_sim · condor_sim · intraday_sim → SQLite ledgers
 L4  Research engine   metrics · regime · factors · portfolio_analyzer · risk_analytics ·
                       attribution · data_quality → results/*.json
 L5  Presentation      Flask dashboard (:5050, read-only console) · digest.py
 ──  Side branch       Kite (Zerodha) read-only: login · exchange · kite_client · watch
```

Flow is unidirectional; the dependency graph is acyclic and layered.

**Research engine (all lenses, not tradeable strategies):** performance metrics (Sharpe/Sortino/Calmar, walk-forward, Monte Carlo), market-regime classifier, 16-factor cross-sectional library, multi-factor ranker (`core4` default), constraint-aware portfolio optimizer, Risk Engine v2 (downside monitor, no orders), Indian market intelligence (expiries, corp actions, surveillance, sectors), daily research-automation pipeline, and a rule-based read-only AI research assistant.

### Methodology (held throughout)
